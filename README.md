# opeclesson.github.io
هذا الموقع عبارة عن منصة تعليمية تفاعلية باللغة العربية لدرس منظمة الدول المصدرة للنفط “أوبك”، يقدّم اختباراً متعدد الأنماط يشمل أسئلة اختيار من متعدد، صح وخطأ، ووصل، مع عرض النتيجة النهائية وتصحيح الإجابات. كما يتضمن أنشطة تعليمية ممتعة مثل توقع الحروف الناقصة والكلمات المتقاطعة، إضافة إلى إمكانية تعديل الأسئلة وإضافة أسئلة جديدة.
